# SaduovWork8
✅ Описание проекта

Quest Master — это полноценное настольное GUI-приложение на PyQt6 для автоматизированного создания, редактирования и экспорта квестов.
Приложение реализует:

генерацию квестов через удобный визуальный интерфейс;

работу с шаблонами и экспортом документов (PDF, DOCX);

графический редактор карт на QPainter/QGraphicsView;

встроенную систему версий и базу данных SQLite;

геймификацию (XP, уровни, достижения).

Задание выполнено в полном соответствии с требованиями.
